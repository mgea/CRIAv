## Prácticas 

Programa de prácticas de asignatura Creatividad e Innovación Audiovisual-B. 

<- [Volver a menu principal](https://github.com/mgea/criav)  

### Objetivo de las prácticas 

El Reto de las prácticas es:
*  el **desarrollo de PROYECTO CREATIVO** (UNA IDEA DE PRODUCCIÓN AUDIOVISUAL) **para la WEB**
*  usando **diferentes herramientas y tecnologías de producción de contenidos**
*  basado en una **estrategia de creación colectiva** (en EQUIPO)
*  y **documentando el proceso en repositorios abiertos** de crowdsourcing
